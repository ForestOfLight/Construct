> [Download **Canopy** here!](https://github.com/ForestOfLight/Canopy)

## Usage

All commands are prefixed with `./`. Do `./help` for more information.

**Usage: `./struct add <name>`**
Adds a new structure.

**Usage: `./struct remove <name>`**
Removes a structure.
